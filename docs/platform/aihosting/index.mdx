# Using AI models on the mittwald platform

The mittwald platform offers a variety of AI models that you can integrate into your applications to enhance functionality and user experience. These models provide capabilities such as natural language processing and (in the future) image generation capabilities, allowing you to build intelligent features without the need for extensive machine learning expertise. This section guides you through the available AI models, how to access them, and best practices for implementation in your projects.
